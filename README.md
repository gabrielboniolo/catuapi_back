# ☕ Catuapi

Full stack system for specialty coffee management, built as an MVP for the **Basic Full Stack Development** course — **Software Engineering | PUC-Rio**.

---

## 📁 Structure

| Directory | Description |
|---|---|
| [`api/`](./api) | Backend — REST API built with Flask, SQLAlchemy and SQLite. Exposes CRUD endpoints for coffee records and includes interactive Swagger documentation |
| [`coffee_signup_form/`](./coffee_signup_form) | Frontend — Single Page Application in plain HTML, CSS and JavaScript that consumes the Flask API |

Refer to the `README.md` inside each directory for setup and execution instructions.
